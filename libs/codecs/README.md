A collection of decoders (and eventually encoders) for working with [Sierra On-line][sierra] [SCI-engine][sci0] assets.

## Documentation

Full documentation for the library can be found [here](https://32bitkid.github.io/sci.js/modules/_4bitlabs_codecs.html).

| Name                    | Decoder | Encoder |
| ----------------------- | :-----: | :-----: |
| [Huffman][huffman]      |   ✅    |         |
| [Lempel–Ziv–Welch][lzw] |   ✅    |         |
| COMP3                   |   ✅    |         |
